# Vehicle map icons (35×35 PNG)

Put your vehicle PNG images in **this folder**. The map uses them for driver markers.

## Required files

| Save as    | Used for      |
|-----------|----------------|
| **car.png**  | Mini & Sedan   |
| **bike.png** | Bike           |
| **auto.png** | Auto (rickshaw)|

Recommended size: **35×35 pixels** (or larger; the app scales to 35×35).

## How to add your images

1. **In Cursor / VS Code**
   - Open the file explorer (left sidebar).
   - Go to: `frontend` → `public` → `assets` → `icons`.
   - Drag and drop your image files into the `icons` folder.
   - Rename them to `car.png`, `bike.png`, and `auto.png` if needed.

2. **In File Explorer (Windows)**
   - Open: `c:\Users\AAHIL\Desktop\Uber-Clone\frontend\public\assets\icons\`
   - Copy or move your images into this folder.
   - Use the exact names: `car.png`, `bike.png`, `auto.png`.

3. **If you want me to use your images**
   - You can paste images in the chat and say which is car, bike, and auto; I can then tell you the exact steps or file names.
   - Or place the files in this folder with the names above and refresh the app.

If a file is missing, the map uses built-in SVG icons for that vehicle type until you add the PNG.
